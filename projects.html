Multiplayer Battle City–Inspired Tank Game

Unity Photon PUN Multiplayer Android Clean Architecture

Overview:

A real-time multiplayer 2D tank battle game inspired by classic Battle City, supporting 1–4 players with lobby, room system, ready-check, synchronized spawning, and smooth gameplay.

Key Contributions:

  • Designed multiplayer lobby, room creation & player ready system
  • Implemented Photon room-based architecture with Auto Sync Scene
  • Built clean MVP-based UI architecture
  • Developed modular enemy AI using State & Strategy patterns
  • Optimized network data flow to reduce sync overhead

Challenges & Solutions:

Challenge: Keeping gameplay synchronized when multiple players join at different times.
Solution: Implemented a ready-state system using Photon player properties and synced scene loading once all players were ready.

Network Data Optimization Demo

Overview:

A technical demo showcasing how optimized network data structures significantly reduce bandwidth usage while maintaining smooth remote player movement.

Key Contributions:

Why This Matters:

This project demonstrates deep understanding of real-time networking constraints and performance-aware engineering.
