Project Overview

Alnahsha Run is a large-scale endless runner themed around iconic cities such as Riyadh, Dubai, and Cairo. The game has crossed 5M+ downloads across Android and iOS.

I worked as a Unity Developer on the live production team at Deftsoft, contributing to gameplay improvements, UI systems, optimization, and live bug-fix cycles.

My Role & Responsibilities

  • Developed reusable UI systems for scalable city-based content
  • Implemented dynamic UI scaling for multiple screen sizes
  • Made UI layouts localization-ready for global markets
  • Worked on gameplay improvements and production bug fixes
  • Collaborated using Git-based workflows with designers and engineers

Engineering & System Design

Reusable UI Scaling System

Designed a flexible UI scaling system that adapted cleanly across multiple aspect ratios without manual fixes.

Localization-Ready Architecture

Structured UI layouts to support text expansion and RTL languages without breaking visual hierarchy.

Live Game Optimization

Assisted with performance and stability improvements during live production updates impacting millions of users.

Challenges & Solutions

Challenge: Maintaining UI consistency across a wide range of devices and languages.

Solution: Designed scalable layouts and reusable UI rules that minimized regression during live updates.

Impact & Results

  • Improved UI consistency across Android & iOS devices
  • Reduced UI-related regressions in live builds
  • Enabled faster rollout of new themed content
  • Contributed to stability of a 5M+ user production game
