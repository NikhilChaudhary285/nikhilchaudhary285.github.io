Project Overview

Alnahsha Run is a large-scale endless runner game themed around iconic cities such as Riyadh, Dubai, and Cairo, designed for a global mobile audience. The game has crossed 5 million downloads across Android and iOS platforms.

I worked on this project as part of the production Unity team at Deftsoft, contributing to gameplay improvements, UI systems, optimization, and live bug-fix cycles.

My Role & Responsibilities

  • Collaborated on player experience improvements and UI systems
  • Designed reusable UI tools for city-themed level content
  • Built a dynamic UI scaling system for multiple screen sizes
  • Made UI fully localization-ready for global audiences
  • Assisted with gameplay feature improvements and bug fixing
  • Participated in optimization and live production bug-fix cycles
  • Collaborated using Git-based workflows with designers and developers

Engineering & System Design

Reusable UI Scaling System

One of my key contributions was designing a reusable UI scaling system that allowed UI layouts to adapt consistently across a wide range of mobile screen sizes and aspect ratios.

  • Reduced manual UI fixes during content updates
  • Improved visual consistency across devices
  • Enabled faster iteration for new city themes

Localization-Ready UI

I helped restructure UI layouts to support localization, ensuring text expansion and right-to-left languages could be handled without breaking layouts.

Live Game Optimization

Worked on performance and stability improvements during live production cycles, ensuring UI updates and gameplay changes did not introduce regressions for millions of active users.

Challenges & Solutions

Challenge: UI consistency across diverse devices

Supporting a global user base meant handling a wide range of resolutions, aspect ratios, and language lengths.

Solution: Designed flexible layouts, scalable UI rules, and reusable UI components that could be safely reused across themes and updates.

Impact & Results

  • Improved UI consistency across multiple devices
  • Reduced UI-related bugs during live updates
  • Enabled faster content rollout for new city themes
  • Contributed to stability of a live game with 5M+ users
